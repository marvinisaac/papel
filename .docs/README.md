## Papel Documentation

- **Location**: All design documents, specs, and planning docs for Papel live in this `.docs/` directory.
- **Scope**:
  - Architecture overviews and deeper technical design.
  - Phase-by-phase specs (MVP, backend, Git integration, UX polish, advanced features).
  - Any future RFCs or decision records related to Papel.
- **Relationship to `README.MD`**:
  - `README.MD` provides a high-level overview for new readers.
  - `.docs/*.md` files contain evolving, detailed documentation that can grow without cluttering the root README.
